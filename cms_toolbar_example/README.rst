===================
CMS Toolbar Example
===================

This is an example for the CMS toolbar addon.

Usage
=====

On your odoo website go to `/cms-toolbar-example`.
By default the last demo record is showed.
You can delete/publish/unpublish.
`edit` and `create` buttons are faked since there's no form to edit demo record.
